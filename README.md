# KeyboardLanguageFixer

A small macOS utility that fixes text typed in the wrong keyboard layout.

If you’ve ever typed a sentence only to realize your keyboard was set to the wrong language, this app converts the text to the correct layout instantly by mapping keys based on their physical keyboard positions.

## Features

* Fix text typed with the wrong keyboard layout
* Works between multiple keyboard layouts
* Fast and simple
* Native macOS / iOS multi platform app built with Swift

## Supported Keyboard Layouts

* English (US)
* German
* Greek
* Hebrew
* Korean (2-Set)
* Russian
* Ukrainian

## Examples

Typed by mistake (English keyboard intended but Hebrew layout active):

```id="ex1"
akuo
```

Converted text:

```id="ex2"
שלום
```

Typed by mistake (Russian layout active but English intended):

```id="ex3"
руддщ
```

Converted text:

```id="ex4"
hello
```
